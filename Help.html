How to upload image to gallery

1.Click on "Add photo" button in the landing page.

2.In first input write URL of image you want to upload

3.In second input add a description of the image

4.Finally press "upload" button to upload photo

How to get image URL

To get an image URL using Chrome, Firefox or Safari, right-click on the image you would like to use, and select either "Copy Image Location", "Copy Image URL" or "Copy Image Addres" from the right-click menu. Then paste the URL into the TinEye search field by right-clicking in the search field and selecting "Paste".

Using Internet Explorer, right-click on the image you would like to use, and select "Properties" from the right-click menu. In the Properties window, highlight the URL and copy it by selecting right-click > "Copy" or using Ctrl+C. Then paste the URL into the TinEye search field by right-clicking in the search field and selecting "Paste".
